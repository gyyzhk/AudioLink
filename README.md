# AudioLink 语音传输系统

## 项目介绍

AudioLink 是一款局域网语音传输系统，可以将手机麦克风音频实时传输到电脑端。

## 版本：v2.2

## 功能特点

- 实时语音传输
- 支持TCP/UDP/HTTP协议
- 支持PCM/OPUS音频编码
- 录音质量选择（高/标准/低音质）
- 录音文件自动保存为WAV格式

## 技术规格

- 采样率: 16000 Hz (可调)
- 声道: 单声道
- 位深: 16 bit
- 端口: 8080
